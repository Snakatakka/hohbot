# HOHbot Overview

HOHbot is a simple discord bot i designed and programmed so that I could both get better at writing code for things that aren't video games* and to make it easier to host camps and ORGs (online reality games). while HOHbot doesn't have very many fuctions / practical use cases, it does (or at least will in the future) make it a lot easier and simpler for me to host my ORG.

Currently, HOHbot only works when im actively running the code, because i'm too lazy / cheap / inexperienced to find a good place to host it.
